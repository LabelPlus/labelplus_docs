# LabelPlus 帮助文档

LabelPlus\(标号器+\)是一款免费开源的漫画翻译辅助工具, 目标为翻译者提供工作平台, 并减少翻译工作交接成本.

此外, 使用额外的PS脚本可以将LabelPlus中编辑的翻译文本直接导入到Photoshop工作环境中.
